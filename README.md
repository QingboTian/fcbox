# 1. 项目简介
项目本身没有什么技术含量，单纯为了方便日常生活而已，不用太在意项目结构以及代码是否优雅，本身就是为了练习golang的同时写点小工具

网上买的商品却无法送货上门，非常让人苦恼，如果不想去快递柜主动去取，还得打电话联系快递员并且安排送货上门，这个项目就是主动检测丰巢快递柜的快件并联系快递员送货上门，真香！

# 2. 必备依赖
- redis
- 腾讯云短信服务（有限频，若快件过多可能会发送失败）
